# 🏠 House Price Prediction: Machine Learning vs Deep Learning

Predicting residential property prices using both traditional machine learning algorithms and deep learning neural networks.

---

## 📖 Overview

Accurate house price estimation is a classic regression problem in Data Science and Machine Learning.

This project explores two different approaches for predicting house prices:

1. 🤖 Traditional Machine Learning Models
2. 🧠 Deep Learning Neural Networks (Keras/TensorFlow)

The primary objective is to compare how different modeling techniques perform on the same housing dataset.

---

## 🎯 Project Goals

- Understand relationships between housing features and prices
- Build predictive regression models
- Compare Machine Learning and Deep Learning approaches
- Evaluate model performance using regression metrics

---

## 📊 Dataset Features

Typical housing attributes include:

| Feature | Description |
|----------|------------|
| Avg Area Income | Average area income |
| Avg Area House Age | Average house age |
| Avg Area Number of Rooms | Average rooms |
| Avg Area Number of Bedrooms | Average bedrooms |
| Area Population | Area population |
| Price | Target variable |
